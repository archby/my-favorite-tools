# **my favorite tools**

list of my favorite tools, websites, software, repos and whatever else.
* [DistroSea](https://distrosea.com/) - Lets you test Linux distros without actually downloading them.
* [LFS](https://linuxfromscratch.org/) - Make your own Linux.
* [ProtonDB](https://www.protondb.com/) - See if Steam games work on Linux.
* [Ventoy](https://ventoy.net/) - Create a bootable USB without flashing the ISO.
* [explainshell](https://explainshell.com/) - Explains what any Linux command does.
* [tldrpages](https://tldr.sh/) - Community driven man pages.
* [AlternativeTo](https://alternativeto.net/) - Open-source alternatives to proprietary software.
* [ArchWiki](https://wiki.archlinux.org/) - Even if you don't use Arch, it's one of the best documentation for Linux.
* [TheFuck](https://github.com/nvbn/thefuck) - Amazing CLI tool that corrects errors in your previous console command when you type "fuck".
* [Lazygit](https://github.com/jesseduffield/lazygit) - Simple TUI for git commands.
* [cheat.sh](https://cheat.sh/) - Cheat sheet for any command.
* [wttr.in](https://github.com/chubin/wttr.in) - TUI, ASCII weather service. Just run 'curl wttr.in'.
* [PrivacyGuides](https://privacyguides.org/) - Community-ran directory dedicated to privacy alternatives to daily software.
* [bat](https://github.com/sharkdp/bat) - 'cat' command but better.
* [zoxide](https://github.com/ajeetdsouza/zoxide) - Better 'cd' command.
* [McFly](https://github.com/cantino/mcfly) - Smarter history search for your terminal.
* [IT-Tools](https://it-tools.tech/) - Beautiful collection of web-based developer utilities.
* [procs](https://github.com/dalance/procs) - Human readable 'ps' command.
* [ModernUnix](https://github.com/ibraheemdev/modern-unix) - List of modern alternatives to Unix/Linux commands.
* [PythonTutor](https://pythontutor.com/) - Visually maps your Java, Python, C/C++ and JavaScript code execution.
* [Diffchecker](https://www.diffchecker.com/) - Intuitive web tool that lets you compare text side by side.
